# HTML変換ツール（業務効率化）

## 概要
ECモール向けHTMLの整形・変換を自動化するためのツール枠です。
実務の運用フロー（手作業→ミス→修正工数）を減らす目的で設計しました。

## 目的
- 定型処理を自動化し、作業時間と入力ミスを削減
- 変換ルールを明文化し、属人化を抑制
- 追加ルールに対応しやすい構造にする

## 想定する入力 / 出力
- 入力：HTML（またはHTML断片）
- 出力：モール仕様に合わせたHTML（整形済み）

## 主な処理（例）
- タグ置換（例：許容されないタグの撤去・置換）
- インラインスタイル / 属性の調整
- テーブル・背景色などの整形
- 文字装飾の統一（環境制約に合わせる）

## 備考
守秘義務のため、実案件のHTMLやクライアント固有情報は含めません。
代わりに、処理方針・ルール設計・運用視点が伝わる形で整理しています。

## CSV Upload Automation

ECモールの商品登録作業を効率化するためのCSV生成ツール。

複数モールへの商品登録作業を効率化することを目的に、
商品データCSVから登録用データやHTMLを自動生成します。

### 主な処理

- 商品データCSV読み込み
- 商品情報の自動展開
- HTMLテンプレート生成
- モール仕様に合わせたデータ整形

**これにより  
商品登録作業の効率化と作業ミス削減を実現しています。**

## Rakuten CSV Scheduled Upload System

楽天RMSの商品登録・更新作業を効率化するためのCSV予約アップロード支援ツール。

商品データCSVを生成し、
楽天RMSのCSVアップロード機能を利用して
**商品登録や更新作業を予約実行できる運用を構築しています。**

### 背景

店舗によっては商品更新作業を翌営業日に回すケースがあり、  
その場合

**深夜〜営業開始までの売上機会を逃す可能性があります。**

そこで

商品更新CSVを事前生成し、  
**深夜帯に予約アップロードする運用**を構築することで

- 商品更新の即時反映
- 売上機会ロスの削減
- 作業負担の軽減

を実現しています。

### 主な処理

- 商品データCSV生成
- 楽天CSV仕様へのデータ整形
- 商品情報の自動展開
- CSVアップロード用データ作成

## Rakuten CSV Scheduled Upload Workflow

楽天RMSの商品更新作業を効率化するための運用改善フロー。

商品データは主にExcelで管理し、  
必要に応じてPythonスクリプトを用いて
CSVデータの生成や一部データの整形を行います。

生成したCSVは楽天RMSのCSVアップロード機能を利用して  
**深夜帯に予約アップロードする運用**を構築しています。

### 運用の目的

店舗によっては商品更新作業を翌営業日に回すケースがあり、  
その場合

**深夜〜営業開始までの売上機会を逃す可能性があります。**

そこで

商品更新CSVを事前生成し  
深夜帯に予約アップロードすることで

- 商品更新の即時反映
- 売上機会ロスの削減
- 作業負担の軽減

を実現しています。

### 使用技術

- Excel
- Python
- CSV処理
- 楽天RMS CSVアップロード
