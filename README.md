# Projects

## 1. ECモールCSVアップロードツール

ECモールの商品CSVを  
FTPアップロードするツール。

### 目的

商品更新時の手動FTP作業を削減し  
作業ミスを防ぐ。

### 機能

- CSVファイルの自動検出
- FTPアップロード
- アップロードログ出力

### 効果

- 手動FTP作業を削減
- 商品更新作業時間を短縮
- アップロードミス防止

### 技術

Python
CSV
FTP


---

## 2. HTML変換ツール（楽天 → 他モール）

楽天の商品ページHTMLを  
Yahooなど別モール用HTMLへ変換するツール。

### 背景

ECモールごとにHTML仕様が異なるため  
商品ページの再編集が必要になる。

この作業を自動化するために開発。

### 機能

- HTML解析
- 不要タグ削除
- モール仕様対応
- 画像URL変換

### 技術

Python
HTML解析
正規表現


### コード量

約200行

---

## 3. 動画圧縮ツール

EC商品動画の容量を  
自動圧縮するツール。

### 背景

商品動画の容量が大きいと

- アップロード制限
- ページ表示速度低下

が発生するため。

### 機能

- 動画ファイル自動検出
- ffmpegによる圧縮
- 圧縮ログ出力
- 圧縮後ファイル保存

### 技術

Python
ffmpeg
動画処理
