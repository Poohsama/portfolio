# Featured Projects

## 1. CSV Upload Scheduler (.NET + SFTP)

### 概要

楽天などの更新作業を指定時刻に自動実行する予約アップロードツール。

既存Python運用を .NET GUI化することで、
Python未導入環境でも利用可能な構成へ改善しました。

### 解決した課題

- 更新時間まで待機が必要
- 特定担当者依存になりやすい
- Python環境構築が必要
- 手動実行によるミス

### 実装内容

- C# / .NET GUI
- YAML設定管理
- SFTPアップロード
- CSV複数管理
- 予約実行
- エラーログ管理

### 想定効果

- 更新待機時間を月約7〜8時間削減見込み
- 年間約90時間の工数削減想定
- 手動実行による人的ミス低減
- 属人化しやすい更新作業を標準化
