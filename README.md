# EC Operations Automation Portfolio

ECサイト運用の業務改善ツールを開発しています。  
主にPythonを用いた自動化ツールを作成し、EC運用の効率化と作業ミス削減を目的としています。

---

# Main Project

## Rakuten CSV Scheduled Upload Workflow

楽天市場では、商品情報更新のためにCSVアップロードを行います。

通常の運用では

・営業開始後にCSV更新  
・手作業でアップロード  
・更新タイミングが遅れる  

といった問題がありました。

特に店舗によっては翌営業日に対応する場合があり、  
深夜〜営業開始までの売上機会を逃す可能性があります。

この問題を解決するため、

**CSV予約アップロードのワークフローを構築しました。**

### Features

・CSV更新の予約アップロード  
・深夜更新による売上機会ロス防止  
・手作業アップロード削減  
・更新作業の安定化

# Other Automation Tools

## HTML Conversion Tool

楽天の商品ページHTMLを  
au PAYマーケット向けHTMLへ変換するツール。

モールごとにHTML仕様が異なるため、  
手作業変換では時間とミスが発生していました。

このツールにより

・HTML変換作業の自動化  
・作業時間削減  
・人的ミス削減  

を実現しました。

---

## Image Placement Automation

au PAYマーケットの商品画像設置を効率化するツール。

手作業で画像設置を行う場合

・作業時間が長い  
・セッションタイムアウト  
・アップロード失敗

といった問題がありました。

ツール化により

・画像設置作業の高速化  
・作業安定化  

を実現しました。

---

## EC Video Compression Tool

EC商品ページ用の動画を最適化する動画圧縮ツール。

商品動画は容量が大きくなりやすく

・アップロード時間増加  
・ページ表示速度低下  

といった問題があります。

FFmpegを利用して

・動画容量削減  
・アップロード時間短縮  
・ページ表示速度改善  

を実現しました。
