# Main Tool

## HTML Converter

ECモールごとのHTML仕様の違いを吸収するためのツール。

楽天の商品ページHTMLを元に  
Yahooショッピング / au PAYマーケット向けHTMLを生成します。

### 主な処理

- 不要タグ削除
- モール仕様の差分調整
- 画像URL調整
- HTML構造整理

これにより

**商品ページ制作作業の時間短縮と  
ミス削減を実現しています。**

---

# Background

ECサイト運用では  
同一商品ページを複数モールに展開する作業が多く発生します。

この作業は

- 手作業が多い
- ミスが起きやすい
- 作業時間が長い

という課題があります。

そこで  
HTMLを自動変換するツールを作成し  
作業時間の短縮と作業ミス削減を目的とした仕組みを構築しました。
