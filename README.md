# Featured Projects

## 1. Multi-Mall Sales Aggregation Tool

### 概要

複数ECモールの売上CSVを統合し、
Excelテンプレートへの転記から親品番集計・SKU集計までを自動化する業務支援ツール。

モールごとに異なるCSV形式を吸収し、日次・月次集計業務の効率化と標準化を目的として開発しました。

対応モール

* 楽天市場
* Yahoo!ショッピング
* au PAY マーケット
* Amazon
* Qoo10
* dショッピング

### 解決した課題

* モールごとにCSV形式が異なる
* 手作業による転記ミス
* 集計作業に多くの時間がかかる
* 特定担当者へ依存しやすい運用
* ヒューマンエラー発生リスク

### 実装内容

* Python

  * 集計処理全体の実装

* pandas

  * CSVデータの読込・変換

* openpyxl

  * Excel転記・オートフィル処理

* YAML

  * モールごとの差分設定管理

* tkinter

  * 非エンジニア向けGUI作成

* logging

  * 実行結果・エラー内容の記録

* 親品番集計機能

  * 商品単位で売上を集計

* SKU集計機能

  * 商品バリエーション単位で売上を集計

### 設計判断

* 売上データのため完全自動化は採用しない
* 最終確認は人間が行う
* サイレントエラーを防止
* 成功・失敗を問わずログを出力
* CSV取り違えによる事故を防止
* モール追加を考慮してYAML化
* 非エンジニアでも利用できる操作性を重視

### 想定効果

* 日次集計時間 約25分 → 約3分へ短縮見込み
* 月40時間規模の作業削減見込み
* 転記ミス削減
* 集計作業の標準化
* 属人化の解消

## 2. CSV Upload Scheduler (.NET + SFTP)

### 概要

楽天などの更新ファイルを指定時刻に自動アップロードする予約実行ツール。

既存Python運用をWindowsアプリ化し、
Python未導入環境でも利用できる構成へ改善しました。

### 解決した課題

* 更新時間まで待機する必要がある
* 担当者によって運用が属人化しやすい
* Python環境構築が必要
* 手動実行によるミスが発生する
* 実行忘れのリスクがある

### 実装内容

* C#

  * アプリケーション実装

* .NET

  * Windowsデスクトップアプリ化

* WinSCP

  * SFTPアップロード処理

* YAML

  * 設定ファイル管理

* GUI

  * 非エンジニア向け操作画面

* 予約実行機能

  * 指定時刻での自動アップロード

* ログ出力

  * 実行結果・エラー内容の記録

### 想定効果

* 更新待機時間を月約7〜8時間削減見込み
* 年間約90時間の工数削減想定
* 実行忘れ防止
* 手動作業による人的ミス削減
* 更新作業の標準化

## 4. Local AI Coding Assistant

### 概要

社内コードや機密情報を外部へ送信せず利用できる開発支援ツール。

ローカル環境のみで動作するため、
セキュリティを確保しながら開発効率向上を目指しました。

### 解決した課題

* 機密コードを外部サービスへ送信できない
* API利用料が継続的に発生する
* 修正作業の初動に時間がかかる

### 実装内容

* Python

  * アプリケーション実装

* PySide6

  * GUI作成

* Ollama

  * ローカルAI実行環境

* コード生成支援

  * 定型コード作成補助

* 修正提案機能

  * 改善案の提示

* レビュー支援

  * コード確認作業の補助

### 想定効果

* コード修正初動時間短縮
* 外部送信不要によるセキュリティ向上
* API費用不要
* ローカル環境のみで継続利用可能
