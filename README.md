# Featured Projects

## 1. .NET CSV予約アップロードシステム
### 概要
CSVファイルを指定時刻にアップロード予約できるシステム。
Python未導入の端末でも利用でき、現場配布を前提とした構成に改善しました。

### 解決した課題
- 夜間更新のための手動待機
- 特定担当者しか扱えない属人化
- 実行ミス、設定ミス
- Python環境依存

### 実装ポイント
- C# / .NET（GUI）
- YAMLによる設定管理
- 予約時刻指定
- CSVファイル管理
- エラー通知確認
- 非エンジニアでも扱いやすい画面設計

### 業務上の価値
単なる自動化ではなく、
**「Pythonがなくても回る」「担当者が変わっても使える」**
状態にしたことで、運用の安定化と引き継ぎ性を高めました。

## 3. EC動画圧縮ツール
### 概要
商品動画を一括圧縮し、アップロード負荷と表示負荷を下げるツール。

### 解決した課題
- 動画容量過多
- アップロード時間の増加
- ページ表示速度低下

### 主な処理
- 動画自動検出
- ffmpegによる圧縮
- 圧縮ログ出力
- 出力管理

### 使用技術
- Python
- ffmpeg
