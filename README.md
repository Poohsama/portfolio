# Featured Projects

## 1. .NET CSV Upload Scheduler

### 概要

Python未導入PCでも利用できる、CSV予約アップロードツール。  
既存Python運用を .NET 化し、配布性と継続運用性を改善しました。

### 解決した課題

- 更新時間まで待機が必要
- Python環境が必要
- 特定担当者依存になりやすい
- 手動実行ミスのリスク

### 実装内容

- C# / .NET GUIアプリ
- CSV複数管理
- 指定時刻アップロード
- YAML設定管理
- Windows端末で利用可能

### 成果

- 属人化しやすい更新作業を標準化
- 他端末展開しやすい構成へ改善
- 夜間更新作業の負担軽減

## 2. Local AI Coding Assistant

### 概要

ローカルLLM（Ollama）を活用した、  
セキュア環境向けコーディング補助ツール。

### 解決した課題

- 機密コードを外部AIへ送れない
- API課金依存
- 修正作業の初速不足
- ローカル開発支援環境不足

### 実装内容

- Python
- PySide6 GUI
- Ollama連携
- 新規コード生成
- 修正提案
- レビュー補助

### 成果

- ローカル環境のみでAI補助開発可能
- セキュリティ意識を保った活用モデルを構築
