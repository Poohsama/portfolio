# Featured Projects

## 1. .NET CSV Upload Scheduler

### 概要

CSVファイルを指定時刻にアップロード予約できるWindows向けツール。

Python未導入PCでも使えるよう .NET で再構築し、配布性と運用性を改善しました。

### 解決した課題

- 夜間更新のための手動待機
- Python環境が必要
- 特定担当者しか扱えない
- 設定ミスによる更新事故

### 実装内容

- C# / .NET
- GUI操作画面
- YAML設定管理
- CSVファイル登録
- 予約時刻実行
- エラー確認運用

### 成果

- 属人化しやすい更新作業を標準化
- 他PCでも利用可能に改善
- 引き継ぎコスト削減

## 3. EC Video Compression Tool

### 概要

- 出力先管理

### 使用技術

- Python
- ffmpeg
