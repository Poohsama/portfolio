# Mak.K

## Workflow Automation / Business Process Improvement

業務改善・運用設計・自動化を目的として開発したプロジェクトをまとめています。

私はPythonを使うことが多いですが、目的はプログラムを書くことではありません。

現場で発生する手作業・待機時間・ヒューマンエラー・属人化といった課題を分析し、継続運用できる仕組みへ改善することを重視しています。

業務フローの整理からツール設計・実装・運用まで一貫して行い、「誰でも使える仕組み」を作ることを意識して開発しています。

# Featured Projects

## 1. Multi-Mall Sales Aggregation Tool

### 概要

複数ECモールの売上CSVを統合し、Excelテンプレートへの転記から親品番集計・SKU集計までを自動化する業務支援ツールです。

モールごとに異なるCSV形式を吸収し、日次・月次集計業務の効率化と標準化を目的として開発しました。

### 対応モール

* 楽天市場
* Yahoo!ショッピング
* au PAY マーケット
* Amazon
* Qoo10
* dショッピング

### Before

* モールごとにCSV形式が異なる
* 手作業による転記ミスが発生する
* 集計作業に時間がかかる
* 特定担当者へ依存しやすい
* ヒューマンエラー発生リスクが高い

### After

* CSVを自動読込
* モールごとの差異を吸収
* Excelテンプレートへ自動転記
* 親品番単位で自動集計
* SKU単位で自動集計
* ログによる処理結果確認

### 実装内容

#### Python

* 集計処理全体の実装

#### pandas

* CSVデータの読込・変換

#### openpyxl

* Excel転記・オートフィル処理

#### YAML

* モールごとの差分設定管理

#### tkinter

* 非エンジニア向けGUI作成

#### logging

* 実行結果・エラー内容の記録

### 設計判断

* 売上データのため完全自動化は採用しない
* 最終確認は人間が行う
* サイレントエラーを防止
* 成功・失敗を問わずログを出力
* CSV取り違えによる事故を防止
* モール追加を考慮してYAML化
* 非エンジニアでも利用できる操作性を重視

### 実際の効果

* 日次集計時間 約25分 → 約3分へ短縮
* 月40時間規模の作業削減見込み
* 転記ミス削減
* 集計作業の標準化
* 属人化の軽減

## 2. CSV Upload Scheduler (.NET + SFTP)

### 概要

楽天などの更新ファイルを指定時刻に自動アップロードする予約実行ツールです。

既存Python運用をWindowsアプリ化し、Python未導入環境でも利用できる構成へ改善しました。

### Before

* 更新時間まで待機する必要がある
* 実行忘れが発生する
* 手動実行によるミスが起きる
* Python環境構築が必要
* 担当者によって運用が属人化しやすい

### After

* 指定時刻で自動実行
* 実行忘れを防止
* GUI操作のみで利用可能
* Python未導入環境で利用可能
* ログによる結果確認

### 実装内容

#### C#

* アプリケーション実装

#### .NET

* Windowsデスクトップアプリ化

#### WinSCP

* SFTPアップロード処理

#### YAML

* 設定ファイル管理

#### GUI

* 非エンジニア向け操作画面

#### 予約実行機能

* 指定時刻での自動アップロード

#### ログ出力

* 実行結果・エラー内容の記録

### 実際の効果

* 更新待機時間を月約7〜8時間削減
* 年間約90時間の工数削減
* 実行忘れ防止
* 人的ミス削減
* 更新作業の標準化

## 4. Local AI Coding Assistant

社内コードや機密情報を外部へ送信せず利用できるローカルAI開発支援ツールです。

ローカル環境のみで動作するため、セキュリティを確保しながら開発効率向上を目指しました。

* 修正作業の初動に時間がかかる

### After

* 外部送信不要
* API費用不要
* コード修正支援を高速化

### 実装内容

#### Python

* アプリケーション実装

#### PySide6

* GUI作成

#### Ollama

* ローカルAI実行環境

#### コード生成支援

* 定型コード作成補助

#### 修正提案機能

* 改善案の提示

#### レビュー支援

* コード確認作業の補助

### 実際の効果

* コード修正初動時間短縮
* 外部送信不要によるセキュリティ向上
* API費用不要
* ローカル環境のみで継続利用可能
