# Featured Projects

## 1. Multi-Mall Sales Aggregation Tool

### 概要

複数ECモールの売上CSVを統合し、
Excelテンプレートへの転記から親品番集計・SKU集計までを自動化する業務支援ツール。

モールごとに異なるCSV形式を吸収し、日次・月次集計業務の効率化と標準化を目的として開発しました。

対応モール

* 楽天市場
* Yahoo!ショッピング
* au PAY マーケット
* Amazon
* Qoo10
* dショッピング

### 解決した課題

* モールごとにCSV形式が異なる
* 手作業による転記ミス
* 集計作業に多くの時間がかかる
* 特定担当者へ依存しやすい運用
* ヒューマンエラー発生リスク

### 実装内容

* Python
* pandas
* openpyxl
* YAMLマッピング管理
* GUI
* CSV自動取込
* Excel自動転記
* オートフィル処理
* 親品番集計
* SKU集計
* ログ出力
* モール判定機能

### 設計判断

* 売上データのため完全自動化しない
* 最終確認は人間が実施
* サイレントエラーを防止
* 成功・失敗を問わずログ出力
* CSV取り違えを防止
* モール追加を考慮してYAML化
* 非エンジニアでも利用できるGUIを採用

### 想定効果

* 日次集計時間 約25分 → 約3分へ短縮見込み
* 月約40時間規模の作業削減見込み
* 転記ミス削減
* 集計作業の標準化
* 属人化の解消

## 2. CSV Upload Scheduler (.NET + SFTP)

### 概要

楽天などの更新作業を指定時刻に自動実行する予約アップロードツール。

既存Python運用を .NET GUI化することで、
Python未導入環境でも利用可能な構成へ改善しました。

### 解決した課題

* 更新時間まで待機が必要
* 特定担当者依存になりやすい
* Python環境構築が必要
* 手動実行によるミス

### 実装内容

* C# / .NET GUI
* YAML設定管理
* SFTPアップロード
* CSV複数管理
* 予約実行
* エラーログ管理

### 想定効果

* 更新待機時間を月約7〜8時間削減見込み
* 年間約90時間の工数削減想定
* 手動実行による人的ミス低減
* 属人化しやすい更新作業を標準化
