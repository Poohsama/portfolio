# Portfolio

ECサイト商品ページ制作と業務効率化ツールのポートフォリオです。

楽天・YahooショッピングなどのECモール向け商品ページ制作と  
EC運用の効率化を目的としたツール開発を行っています。

主に以下の課題解決を目的としています。

- ECモール間の商品ページ展開作業の効率化
- 商品登録作業の自動化・簡略化
- 作業ミスの削減
- 深夜更新による売上機会ロスの削減

### What I Built

**HTML Converter**

楽天商品ページHTMLを  
Yahooショッピング / au PAYマーケット用HTMLへ変換

**CSV Upload Automation**

商品データCSVから  
商品登録用データ・HTMLを自動生成

**Rakuten CSV Scheduled Upload Workflow**

商品更新CSVを深夜帯に予約アップロードする運用を構築  
営業開始前の商品更新を実現し、売上機会ロスを削減

# Main Tools

## HTML Converter

ECモールごとのHTML仕様の違いを吸収するためのツール。

楽天の商品ページHTMLを元に  
Yahooショッピング / au PAYマーケット向けHTMLを生成します。

### 主な処理

- 不要タグ削除
- モール仕様の差分調整
- 画像URL調整
- HTML構造整理
- 文字装飾の統一

これにより

**商品ページ制作作業の時間短縮と  
作業ミス削減を実現しています。**

---

## CSV Upload Automation

ECモールの商品登録作業を効率化するためのCSV生成ツール。

商品データCSVを元に  
商品登録用データやHTMLを自動生成します。

### 主な処理

- 商品データCSV読み込み
- 商品情報の自動展開
- HTMLテンプレート生成
- モール仕様に合わせたデータ整形

これにより

**商品登録作業の効率化と作業ミス削減を実現しています。**

---

## Rakuten CSV Scheduled Upload Workflow

楽天RMSの商品更新作業を効率化するための運用改善フロー。

商品データは主にExcelで管理し、  
必要に応じてPythonスクリプトを用いて
CSVデータの生成や一部データ整形を行います。

生成したCSVは楽天RMSのCSVアップロード機能を利用して  
**深夜帯に予約アップロードする運用**を構築しています。

### 背景

店舗によっては商品更新作業を翌営業日に回すケースがあり、  
その場合

**深夜〜営業開始までの売上機会を逃す可能性があります。**

そこで

商品更新CSVを事前生成し  
深夜帯に予約アップロードすることで

- 商品更新の即時反映
- 売上機会ロスの削減
- 作業負担の軽減

を実現しています。

### 使用技術

- Excel
- Python
- CSV処理
- 楽天RMS CSVアップロード

# Background

ECサイト運用では  
同一商品ページを複数モールに展開する作業が多く発生します。

この作業は

- 手作業が多い
- ミスが起きやすい
- 作業時間が長い

という課題があります。

そこで

HTML変換ツールやCSV生成ツールを作成し、  
作業効率化と運用改善を目的とした仕組みを構築しました。
