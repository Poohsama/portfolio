# Main Project

## CSVアップロード自動化ツール

ECモール向け商品CSVを、指定時刻に自動アップロードするツールです。

### 背景

商品情報の更新は、販売開始タイミングや掲載内容の反映時刻が重要になる一方、
手動アップロードでは夜間対応や作業待機が必要になり、運用負荷とミスの原因になっていました。

### 解決したこと

- CSVアップロードの自動化
- 深夜実行への対応
- 実行時刻の指定
- 手動待機の削減
- 更新タイミングの安定化

### 業務上の価値

単なる作業短縮ではなく、更新タイミングを狙って反映できるようにすることで、
販売機会の確保と機会損失の抑制を目的として開発しました。

### 使用技術

- Python
- CSV
- FTP
- スケジューラ制御
