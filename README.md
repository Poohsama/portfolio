# Portfolio

ECサイトの商品ページ制作と  
業務効率化ツールのポートフォリオです。

楽天・YahooショッピングなどのECモール向け商品ページ制作と  
EC運用の効率化を目的とした自動化ツール開発を行っています。

EC運用では

- 商品ページのモール間展開
- 商品登録作業
- 商品更新作業

など多くの手作業が発生します。

これらの作業には

- 作業時間が長い
- ヒューマンエラーが発生しやすい
- 更新タイミングが遅れる

といった課題があります。

本ポートフォリオでは  
これらの課題を解決するために開発した

**EC運用自動化ツールと運用改善フロー**

を紹介しています。

---

# Main Project

## Rakuten CSV Scheduled Upload Workflow

ECサイトの商品更新を自動化する  
運用改善システムです。

商品データはExcelで管理し、  
Pythonスクリプトを用いてCSVデータ生成や  
データ整形を行います。

生成したCSVはPythonスクリプトから  
SFTP接続を利用して楽天サーバーへアップロードし、  
指定した日時に商品更新を実行する仕組みを構築しました。

これにより

- 深夜帯の商品更新
- 営業開始前の商品反映
- 売上機会ロスの削減
- 更新作業の省力化

を実現しています。

---

### 主な機能

- CSV自動アップロード
- SFTPによる楽天サーバーへの直接転送
- 指定日時でのスケジュール実行
- CSVデータの事前バリデーション
- アップロード失敗時のリトライ処理
- エラー発生時のメール通知

---

### CSVバリデーション

アップロード前にCSVファイルの検証を行い  
不正データによる更新失敗を防止しています。

主なチェック内容

- 空ファイルチェック
- ファイル名ルール確認
- ヘッダー構造確認
- CSV文字コード対応（UTF-8 / Shift-JIS）

これにより

**更新処理の安定化とヒューマンエラー防止を実現しています。**

---

### 使用技術

- SFTP（paramiko）
- CSV処理
- YAML設定ファイル
- SMTPメール通知
- Excelデータ管理

# Other Tools

## HTML Converter

楽天の商品ページHTMLを  
Yahooショッピング / au PAYマーケット用HTMLへ変換するツール。

ECモールごとのHTML仕様の違いを吸収し  
商品ページ制作作業を効率化します。

### 主な処理

- 不要タグ削除
- モール仕様の差分調整
- 画像URL調整
- HTML構造整理
- 文字装飾統一

これにより

**商品ページ制作作業の時間短縮と  
作業ミス削減を実現しています。**

---

## CSV Upload Automation

ECモールの商品登録作業を効率化するための  
CSV生成ツールです。

商品データCSVを元に

- 商品登録用データ
- 商品ページHTML

を自動生成します。

### 主な処理

- 商品データCSV読み込み
- 商品情報自動展開
- HTMLテンプレート生成
- モール仕様に合わせたデータ整形

これにより

**商品登録作業の効率化と作業ミス削減を実現しています。**
